# 🧾 Kashier System

A desktop-based cashier (POS) application designed to streamline daily sales operations for retail shops, factories, and small to medium-sized businesses.

---

## 🚀 Overview
Kashier System is built to manage sales transactions efficiently while providing a simple and user-friendly interface.  
It helps businesses track products, handle invoices, and manage daily operations with ease.

---

## ✨ Features
- 🧾 Create and manage sales invoices  
- 📦 Product management (add, update, delete)  
- 💰 Real-time calculation of totals and payments  
- 🔍 Search and select products easily  
- 📊 Basic reporting for daily sales  
- ⚡ Fast and responsive desktop performance  

---

## 🛠️ Technologies Used
- C#  
- WPF (.NET)  
- MVVM Architecture  

---

## 📦 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Kashier-System.git
   ```

3. Restore NuGet packages

4. Build and run the project
